题目描述：

给定一个整数数组，判断是否存在重复元素。

如果任何值在数组中出现至少两次，函数返回 true。如果数组中每个元素都不相同，则返回 false。

示例 1:

输入: [1,2,3,1]
输出: true
示例 2:

输入: [1,2,3,4]
输出: false
示例 3:

输入: [1,1,1,3,3,4,3,2,4,2]
输出: true

class Solution {
public:
    bool containsDuplicate(vector<int>& nums) {
        //使用map记录每个数出现的次数，如果其中有出现的次数大于1就返回true，否则返回false
        map<int,int>m;
        for(auto&e : nums)
        {
            m[e]++;
            if(m[e] > 1)
                return true;
        }
        return false;
    }
};
